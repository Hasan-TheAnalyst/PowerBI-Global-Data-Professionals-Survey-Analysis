# 📊 Data Professionals Survey Analysis (Power BI Project)

## 📌 Project Overview

This project analyzes a survey of data professionals to uncover insights about **salaries, skills, demographics, and job satisfaction** across different roles in the data industry.

The dashboard was built using **Power BI**, with a focus on transforming raw survey data into meaningful, business-oriented insights.

---

## 🎯 Objectives

The goal of this project is to answer key questions such as:

* 💰 What are the average salaries across different data roles?
* 🌍 How do salaries vary by country and gender?
* 💻 What are the most popular programming languages among data professionals?
* 😊 How satisfied are professionals with their salary and work-life balance?

---

## 🧹 Data Cleaning & Preparation

The dataset required several preprocessing steps before analysis:

* Removed empty and irrelevant columns
* Standardized categorical values (e.g., "Other" responses across multiple columns)
* Split and cleaned text fields for consistency
* Converted salary ranges into a numerical **Average Salary** column

---

## 📊 Dashboard Features

The Power BI dashboard includes:

### 🔹 Demographics

* Distribution of survey participants by country
* Gender-based salary comparison

### 🔹 Salary Insights

* Average salary by job title:

  * Data Scientist
  * Data Engineer
  * Data Analyst
  * Data Architect
  * Database Developer

### 🔹 Skills Analysis

* Most popular programming languages across roles:

  * Python
  * R
  * SQL / Others

### 🔹 Satisfaction Metrics

* Work-life balance score
* Salary satisfaction score

---

## 📈 Key Insights

* 💼 **Data Scientists** tend to have the highest average salaries
* 🐍 **Python** is the most widely used programming language across all roles
* 😊 Work-life balance satisfaction is moderate
* 💰 Salary satisfaction is relatively lower, suggesting potential dissatisfaction with compensation
